Тест по основам Software Testing
Что такое Software Testing?
  • Это работа разработчиков над исправлением багов.
  • Это процесс написания кода для программного обеспечения.
  • Процесс проверки того, что программное обеспечение работает согласно требованиям.
  • Процесс проектирования архитектуры программного обеспечения.
Можно ли проверить все?
  • Да, полный тест возможен для любого проекта.
  • Можно проверить все только в автоматизированном режиме.
  • Да, если написать достаточно тестов, можно проверить все.
  • Нет, полное тестирование невозможно, так как существует слишком много возможных сценариев.
