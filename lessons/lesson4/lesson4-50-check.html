Для чего используется чекбокс (checkbox) в интерфейсе?
  • Для выбора одной из нескольких доступных опций.
  • Для ввода числового значения.
  • Для отображения информации о пользователе.
  • Для выбора одной или нескольких опций, каждая из которых может быть включена или выключена.
